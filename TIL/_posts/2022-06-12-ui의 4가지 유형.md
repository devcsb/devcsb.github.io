# 오늘 배운 것

## User Interface(UI)의 4가지 유형

1. CLI : 텍스트(명령어) 기반
2. GUI : 그래픽 기반 (마우스, 펜)
3. NUI : 신체 부위 이용 (터치, 음성)
4. OUI : 유기적 상호작용 기반 인터페이스 (모든 사물이 입출력장치로 변화)
